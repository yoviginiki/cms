# Slice 0.7: Visual Editor Mode

> **Goal**: Live rendered page view with click-to-select and inline editing.
> **Time cap**: 2 weeks (larger slice)
> **Change cap**: ~60 files

## Acceptance Criteria

1. ☐ Open page editor → default mode is Visual (or last used mode)
   Expected: Page renders as visitor would see it, with subtle edit affordances
   Actual: ___

2. ☐ Hover over a section → subtle highlight/outline appears
   Expected: Light border or overlay showing the section boundary
   Actual: ___

3. ☐ Click a heading module → it becomes selected → settings panel opens
   Expected: Same settings panel as wireframe, heading is highlighted in canvas
   Actual: ___

4. ☐ Click heading text directly in canvas → inline text editing activates
   Expected: Cursor appears in text, can type/edit directly on canvas
   Actual: ___

5. ☐ Type new text inline → text updates in real-time in canvas AND in settings panel
   Expected: Bidirectional sync between inline edit and panel
   Actual: ___

6. ☐ "+" buttons appear between sections for adding new sections
   Expected: Visible insert points when hovering between sections
   Actual: ___

7. ☐ Visual mode renders with correct spacing, colors, backgrounds from design settings
   Expected: Section padding, background color, heading font-size all visible
   Actual: ___

8. ☐ Switch to Wireframe → switch back to Visual → all changes preserved
   Expected: No data loss on mode toggle
   Actual: ___

## Pass condition: ALL 8 criteria PASS
## Fail condition: ANY criterion FAIL → slice not done
