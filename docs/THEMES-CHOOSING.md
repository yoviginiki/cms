# Choosing and customizing a theme

A **theme** is your site's design system — its colors, fonts, spacing, corner
and shadow character, and overall layout personality. Change the theme and every
page restyles at once; the output is plain static CSS, so themes cost nothing at
page-speed.

## Choosing a theme

Open **Themes** in the site sidebar. Each theme shows a **live preview** — a real
render of that theme, not a mockup — so you can see the actual fonts, colors, and
layout before choosing. Click a theme for a full-page preview with a
desktop/mobile toggle, then **Use this theme** to activate it.

### The first-party themes

Five hand-crafted themes ship with every site, each a genuinely different design:

- **Ensō** — the Stillopress flagship: washi (warm paper) monochrome, a single
  vermilion mark, Barlow Condensed over Barlow. Cinematic, quiet, zero radius.
- **Journal** — an editorial magazine: Fraunces serif, drop caps, multi-column
  feature wells, rule-separated stories. Cream, ink, burgundy.
- **Ledger** — minimal business/SaaS: structured grids, stat rows, feature cards,
  a confident blue. Fast-scanning.
- **Atelier** — image-led portfolio: full-bleed galleries, huge project titles,
  monochrome with a single accent.
- **Hearth** — warm lifestyle: terracotta and sage, rounded corners, soft
  shadows, a friendly serif/sans pairing. Recipe- and list-friendly.

## Making your own

Three ways, from lightest to fullest control:

1. **Theme Wizard** — describe a feel, upload a screenshot, or point at a site you
   like, and get an original theme. See *Creating a theme from a site you love*.
2. **Fork + Theme Studio** — fork any theme and edit its tokens visually in Theme
   Studio, with a live preview that matches published output exactly. Forking
   creates an editable copy; it does not change your live site until you activate
   it.
3. **Import** — bring in a theme bundle exported from another site
   (**Import** on the Themes page). Themes export and import as a single
   self-describing file (tokens + metadata), so a design system moves between
   sites cleanly.

## Switching safely

Activating a theme flags every published page and post to be re-published, so the
whole site moves to the new look together (you republish from **Stale pages**).
Per-page theme overrides are possible for one-off landing pages. Overrides you
make while one theme is active stay tied to that theme and won't bleed onto
another.
