Preface

This is a demonstration of the ensodo flipbook library with realistic 3D page turns, curl gradients, and cast shadows.

Chapter 1

The pages are real HTML elements. Links and buttons inside them remain interactive. The content is fully SEO-indexable.

Illustrations

Any block type can be a page — text, images, columns, grids. The flipbook just wraps them in the turn animation.

Try clicking the left or right half of the book to flip.

Chapter 2

The curl gradient shifts based on animation progress, creating a realistic paper shading effect.

Chapter 3

Cast shadows fall on the underlying page, adding depth. Shadow intensity is configurable.

Chapter 4

Cover pages (first and last) animate as rigid planes without curl — like hardback card stock.

Interactive Test

Try this link: Click me — it should fire without triggering a page flip.

Chapter 5

The animation uses requestAnimationFrame for smooth 60fps rendering. All transforms are GPU-compositable.

Responsive

Resize the window below 720px to see single-page mode. The book adapts without destroying and recreating.

Keyboard

Click the book to focus it, then use Arrow keys, PageUp/Down, Home/End to navigate.
