# Universal magazine craft

These rules apply to every issue, every genre. Genre playbooks refine them; they never
override the structural rules marked **hard**.

## Issue anatomy

An issue is a journey with four territories, in order:

1. **Cover** — one page (recto). One image or one typographic statement, the issue title,
   at most three cover lines. The cover sells a single feeling, not the table of contents.
2. **Front-of-book (FOB)** — short items, fast rhythm. Half-page and quarter-page units,
   many entry points per spread. The reader is still deciding whether to commit; give them
   snacks. 2–4 spreads for a small issue.
3. **Feature well** — the long-form heart. Slow rhythm, generous space, the issue's visual
   peaks. Each feature opens on a spread of its own and runs uninterrupted. Order features
   strongest-second: open the well with a good one, put the best one after it, taper toward
   the end.
4. **Back-of-book (BOB)** — columns, essays, service items, the closer. Quieter than the
   well. The last spread is a deliberate ending — a single image, a colophon, a final
   thought — never an abrupt stop.

**Hard:** never open the feature well on the first spread after the cover. The reader needs
a ramp.

## Rhythm

Pacing is the alternation of dense and airy. After a loud spread (full-bleed image, heavy
type) place a quiet one (single column, whitespace). Three dense spreads in a row exhausts;
three airy spreads in a row bores. When planning, read the flatplan as a rhythm strip:
loud–quiet–loud–quiet is a waltz; loud–loud–quiet–loud is syncopation; use both, never
loud–loud–loud.

## Spread thinking

**Hard:** the unit of design is the two-page spread, not the page. Verso and recto are one
canvas the reader sees at once.

Every spread has exactly one dominant element — the thing the eye lands on first. An image,
a headline, a pull quote, a stat. **Hard: never two competing focal points.** If two
elements fight, shrink one until it submits or move it to the next spread.

Secondary elements support the dominant: captions anchor images, standfirsts anchor
headlines, whitespace anchors everything.

## Text–image relationships

An image on a spread is doing one of three jobs. Decide which before placing it:

- **Evidence** — the image proves the text. Place it adjacent to the passage it supports,
  caption it with specifics (who, where, when). Documentary scale: substantial but not
  operatic.
- **Atmosphere** — the image sets temperature. It may bleed, may sit behind text (with a
  gradient overlay for legibility), needs only a minimal caption or none.
- **Interruption** — the image is a pause in a long text. Full-width, breathing room above
  and below, caption optional. Use one per 3–4 pages of continuous text, not more.

Captions carry weight. A caption is the most-read text on the spread after the headline —
write it as a sentence with information in it, never "Photo: description of photo."
**Hard:** no decorative captions, no caption-less evidence images.

## Typography discipline

One display face + one text face per issue. Stillopress default: **Barlow Condensed** for
display, **Barlow** for text. Hierarchy comes from size and weight, not from adding faces.

Working scale (pt): body 9.5–10.5 on 13–14.5 leading; standfirst 13–15; section headers
10–11 caps, tracked; headlines 40–72 display; pull quotes 18–26. Captions 7.5–8.5, one
weight step from body.

Text measure: 45–75 characters per line. In grid terms: body text in 3–4 column spans of
the 12-column grid, never wider than 6 without a deliberate reason (and then leading opens
up).

## Whitespace

Whitespace is material, not leftover. A margin of silence around a headline is part of the
headline. When a spread feels crowded, remove an element — never shrink everything to fit.
A quiet spread after a loud one is not empty; it is the loud one's echo.

## Pull quotes

**Hard: one pull quote per spread, maximum.** Use one only when the quote stands alone —
a reader who sees nothing else should still get something complete. Never pull a sentence
that needs its context; never pull a stat (use a stat treatment instead). Set pull quotes
in display size, give them their own column territory, and keep them away from the passage
they were pulled from.

## Openers

Feature openers get the most visual investment on the spread budget. Two legitimate modes:

- **Image opener** — full-bleed photograph, headline and standfirst placed in its quiet
  zone (sky, shadow, negative space), body text starts on the facing page or below the fold.
- **Typographic opener** — the headline IS the image: display type at poster scale,
  generous whitespace, at most one small supporting image.

**Hard: never both fighting.** A full-bleed image with poster-scale type on top of its busy
area is two openers stapled together. Choose.

## The grid

Every spread respects the 12-column Swiss grid per page (36pt margins, 12pt gutters on an
A4 page). Freedom is how many columns an element spans and where it sits vertically —
never ignoring the grid. Bleeds are allowed past margins to page edges; text never bleeds.
Rotation is a spice: one rotated element per spread at most, and only when the pattern
calls for it.

## Ease-of-use doctrine (for the wizard)

The user is not an editor and does not want to become one. These playbooks exist so the
wizard can make every craft decision itself. When information is missing, choose the most
conventional option and state the choice in one plain sentence — never ask the user a
question a competent editor could answer alone.
